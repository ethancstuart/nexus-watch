# dashview Roadmap

## Shipped
- Panel-based architecture with self-contained data sources
- Dark theme with responsive grid layout
- Weather panel (OpenWeatherMap) with 3-day forecast
- Settings panel with location search and geolocation
- Circuit breaker fetch pattern
- Vercel Edge Function API proxy
- localStorage persistence

## Next Up
- Stocks & Markets panel (Finnhub API — configurable watchlist, sparklines)
- News & RSS panel (server-side RSS proxy, keyword classifier)
- Watchlist and feed configuration in Settings

## Future
- Google Calendar panel (opt-in, OAuth)
- Claude AI Chat panel (opt-in, BYO API key)
- Map panel (MapLibre GL JS)
- Variant system (single codebase, multiple products)
- Keyword monitors and anomaly detection
- Desktop app (Tauri)
- Telegram/notification integration

## Open Source
dashview is MIT licensed and free forever. If community interest grows, a managed hosted version may be offered as a convenience — the open-source tool will always remain fully functional and free.
