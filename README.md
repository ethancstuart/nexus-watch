**Real-time geopolitical intelligence platform.** Earthquakes, wildfires, live flights, cyber threats, news, markets, and AI sitreps -- all on one interactive map with 7 data layers and a full intelligence system. Plus a classic 13-panel personal dashboard.

Built entirely with [Claude Code](https://claude.ai/claude-code). Vanilla TypeScript, no framework. Open source under MIT.

## Intel Map

The primary view -- a full-screen MapLibre GL map with CARTO dark matter tiles and 7 real-time data layers:

| Layer | Source | Visualization | Refresh |
|-------|--------|--------------|---------|
| Earthquakes | USGS | Magnitude-sized dots, depth-colored | 1 min |
| Global News | GDELT | Tone-colored clusters (red=negative, green=positive) | 15 min |
| Wildfires | NASA FIRMS | Heatmap (zoomed out) + point markers (zoomed in) | 10 min |
| Weather Alerts | Open-Meteo | Severity-colored markers at 20 major cities | 30 min |
| Prediction Markets | Polymarket/Kalshi | Probability-colored markers, geocoded by topic | 5 min |
| Live Aircraft | OpenSky Network | Altitude-colored dots with callsign labels | 15 sec |
| Cyber Threats | Cloudflare Radar | Great circle arc lines between threat corridors | 1 hour |

**Intelligence System:**
- **Geo-correlation engine** -- detects earthquake clusters, fire convergence, negative news surges, multi-signal events
- **Country Intelligence Index** -- 23 nations scored 0-100 across events, disasters, sentiment, predictions
- **Intel Bar** -- severity-colored alert pills (critical/elevated/monitor), click to fly to location
- **AI Sitreps** -- on-demand situation reports generated by Claude Haiku from current data
- **Country Panel** -- click any country for stability breakdown with component bars and recent events
- **Widget Overlays** -- open any dashboard panel as a floating, draggable overlay on the map

**Keyboard Shortcuts:** `L` layers, `C` country index, `S` sitrep, `1-7` toggle layers, `?` help

## Classic Dashboard

13 data panels organized into customizable spaces:

- **Weather** -- multi-location forecasts, hourly sparklines (OpenWeatherMap)
- **Markets** -- 10-stock watchlist with real-time quotes (Finnhub)
- **Crypto** -- Top 10 coins with sparklines, market cap (CoinGecko)
- **News** -- 7 categories + custom RSS feeds
- **Sports** -- NBA, NFL, MLB, EPL live scores (ESPN)
- **Entertainment** -- Trending movies/TV (TMDB)
- **AI Chat** -- Multi-provider (Anthropic, OpenAI, Google, xAI)
- **Calendar** -- Google Calendar integration (premium)
- **Notes** -- Quick-capture to-dos, fully offline
- **Globe** -- Interactive 3D news globe
- **Hacker News** -- Top/Best/New/Show/Ask tabs
- **GitHub** -- Activity feed for any username
- **Spotify** -- Now playing + recent tracks (premium)

**AI-Native Interface:** AI Bar (Cmd+K), daily briefings, Pulse Bar intelligence, slash commands.

## Quick Start

```bash
git clone https://github.com/ethancstuart/Dashboard.git
cd Dashboard
cp .env.example .env.local
# Add your API keys to .env.local
npm install
npm run dev
```

Open [localhost:5173/#/intel](http://localhost:5173/#/intel) for the Intel Map, or [localhost:5173/#/app](http://localhost:5173/#/app) for the classic dashboard.

## API Keys

| Service | Tier | Purpose |
|---------|------|---------|
| [OpenWeatherMap](https://openweathermap.org/api) | Free | Weather data |
| [Finnhub](https://finnhub.io) | Free (60 calls/min) | Stock quotes |
| [TMDB](https://www.themoviedb.org/documentation/api) | Free | Entertainment data |

Optional: NASA FIRMS key (wildfire data), Cloudflare Radar token (cyber threats), Stripe keys (premium), Anthropic key (AI sitreps + shell), Spotify client ID, BYO key for AI Chat.

All other intel map data sources (USGS, GDELT, Open-Meteo, Polymarket, OpenSky) are free with no API key required.

## Architecture

```
src/
  map/           # Intel Map engine
    MapView.ts        # MapLibre GL initialization
    MapLayerManager.ts # Layer registry, toggle, persistence
    MapOverlayManager.ts # Floating widget overlays
    layers/           # 7 MapDataLayer implementations
    controls/         # LayerPanel, CountryPanel, ViewToggle
  panels/        # 13 self-contained panel classes
  services/      # API clients, auth, storage, intelligence
  pages/         # Landing, dashboard, intel, roadmap
  ui/            # AI bar, intel bar, settings, widgets
  config/        # Themes, density, preferences
  types/         # Centralized TypeScript types
  styles/        # CSS files using custom properties
api/             # 41 Vercel Edge Functions (API proxies, auth, AI, Stripe)
```

**No framework** -- vanilla TypeScript + DOM. MapLibre GL for maps. Panel classes with lifecycle methods. API keys proxied through Edge Functions.

**Key patterns:** Circuit breaker fetch, priority-based loading, AbortController cleanup, geospatial correlation engine, country stability scoring.

## Tech Stack

- [Vite](https://vitejs.dev) + TypeScript (strict mode)
- Vanilla DOM (no framework)
- [MapLibre GL JS](https://maplibre.org) + CARTO dark matter tiles
- CSS custom properties (JetBrains Mono, terminal aesthetic)
- [Vercel](https://vercel.com) Edge Functions (41 endpoints)
- [Vitest](https://vitest.dev) + happy-dom (104 tests)
- 7 data sources: USGS, GDELT, NASA FIRMS, Open-Meteo, Polymarket, OpenSky, Cloudflare
- Stripe (raw fetch, no SDK) for payments
